# Future Improvements (Next Drill Targets)

This section is how you avoid “DR as a checkbox.” Each item must be actionable and map to a repository change.

## Improvement backlog (example)

1. **Tighten restore preflight for delegated subnets**
   - Action: document explicit delegated-subnet block in preflight checklist
   - Target: next monthly drill

2. **Standardize monitoring ingestion verification queries**
   - Action: include Heartbeat continuity query in the restore drill process doc
   - Target: next monthly drill

3. **Add automated evidence export checklist for reviewers**
   - Action: add “what to export from portal” and “which command outputs to store”
   - Target: next monthly drill

## Limitations

This repository does not implement an automated restore-drill orchestration system.
Improvements should remain operationally light: better checklists, evidence templates, and guardrails.
